# git相关用法
### 初始化仓库
```
    git init
```
### 添加远程仓库地址
```
    git remote add origin "远程仓库地址"
```
### 添加文件
```
    git add .
```
### 提交文件
```markdown
    git commit -m "提交说明"
    //推送到分支
    git push -u origin master
```
### 添加远程仓库
```markdown
    git remote add 远程仓库地址
```
